Introdução à Estrutura de OpenGL

Este laboratório teve como objetivo apresentar um ambiente para programação em linguagem C++ e a biblioteca OpenGL, sendo a mesma escolhida devido à praticidade proporcionada e à grande difusão desta ferramenta no meio acadêmico. Este roteiro foi executado no sistema Ubuntu-linux instalado no laboratório didático.

Instalação da biblioteca FreeGLUT no Ubuntu-linus

A instalação da biblioteca FreeGLUT no Ubuntu-linux deverá ser realizada conforme o mostrado no Laboratório 1.

A Compilação e execução dos programas OpenGL, serão realizadas conforme Laboratório1. Nenhuma IDE foi utilizada, apenas comandos no terminal do Ubuntu-Linux.

  • Exemplo de compilação via linha comando:
    • gcc -o test lesson5.cpp -lglut -lGL -lGLU
    • ./test
